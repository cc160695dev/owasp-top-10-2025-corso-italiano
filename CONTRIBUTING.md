# Come contribuire

Grazie per l'interesse! Sono benvenuti:

- 🐛 **Correzioni** di errori, refusi, link rotti → apri una Issue o una PR diretta
- 🌍 **Traduzioni** di moduli in altre lingue → apri prima una Issue per coordinarci
- 📚 **Aggiornamenti** quando OWASP rilascia revisioni → linkare la fonte
- 🧪 **Nuovi lab pratici** o esercizi → PR con descrizione chiara

## Regole minime

- Mantieni il tono didattico e il formato delle 12 sezioni nei moduli A01–A10.
- Cita sempre le fonti (OWASP, NIST, MITRE ecc.).
- Per gli esempi di codice, includi sia la versione vulnerabile che quella sicura.
- Niente segnalazioni di vulnerabilità reali su sistemi di terzi.
